# Div - Divisão fácil

Sistema para a divisão de gastos em grupos.

[Demo](https://rafael-tcc2-div.vercel.app/)

## Pré-requisitos
- Node.js
- Yarn
- Banco de dados postgresql
- Configurar a váriavel DATABASE_URL no arquivo .env

```bash
yarn
yarn prisma migrate dev
```

## Iniciar
```bash
yarn dev
```

## Tecnologias
- React
- Next.js
- NextAuth.js
- Typescript
- Styled-components
- React Testing Library
- React Hook Forms
- Jest
- Prisma
- Postgresql
